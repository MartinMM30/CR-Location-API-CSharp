# CR Location API - C# & .NET

Una API RESTful desarrollada en C# y .NET orientada a la gestión y consulta de datos geográficos, específicamente enfocada en el manejo y relación de provincias y distritos. Este proyecto demuestra la implementación de una arquitectura basada en controladores y buenas prácticas de desarrollo web.

##  Características Principales

* **Arquitectura de Controladores:** Implementación estructurada de controladores independientes para el Backend (procesamiento de datos) y el Frontend (consumo y presentación).
* **Data Binding:** Resolución y enlace de datos relacionales precisos entre las entidades geográficas (Provincias y Distritos).
* **Documentación Interactiva:** Integración nativa con **Swagger** (OpenAPI) para la exploración, prueba y documentación clara de todos los endpoints de la API.
* **Programación Orientada a Objetos:** Código estructurado bajo principios sólidos de POO.

##  Stack Tecnológico

* **Lenguaje:** C#
* **Framework:** .NET 
* **Documentación / API Testing:** Swagger UI
* **Arquitectura:** Controladores API

##  Cómo ejecutar el proyecto localmente

1. Clonar este repositorio en tu máquina local.
2. Abrir el archivo de la solución (`.sln`) utilizando Visual Studio.
3. Compilar y ejecutar el proyecto (F5).
4. El navegador abrirá automáticamente la interfaz de Swagger, permitiendo interactuar directamente con los servicios web.
